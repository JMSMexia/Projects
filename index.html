HTML tables

A table is a great way to show information, so it something that every developer should lear to do in html.
Below you can find a table showing the main tags related to tables:

tag Denition Use
<table> and </table> Table delimiter Defines the begining and the end of a table.
<thead> and </thead> Table Header delimiter Defines the begining and the end of the header of the table.
<th> and </th> Table header cell data delimiter. Inserts data into a cell and defines it as header data.
<tr> and </tr> Table row delimiter Inserts a new row.
<td> and </td> Table cell data delimiter Inserts data into a cell.
<tfoot> and </tfoot> Table Footer delimiter Defines the begining and the end of the footer of the table.

HTML Sectioning tags

Using correct sectioning tags is the best way to mae sure your browser and SEO functions properly and enhances your website visits.
Find below the semantics sectioning tags:

tag Definition
<address> Identifies da contect as the contact information for a person or people, or an organizations.
<article> Identifies a self-contained composition in a document, page, application, or site, which is intended to be independently distributable or reusable (e.g., in syndication).
<aside> Represents a portion of a document whose content is only indirectly related to the document's main content. Asides are frequently presented as sidebars or call-out boxes.
<footer> Represents a footer for its nearest ancestor
<header> Represents introductory content, typically a group of introductory or navigational aids.
<h1> to <h6> Represent six levels of section headings.
<main> Represents the dominant content of the body of a document.
<nav> Represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.
<section> Represents a generic standalone section of a document.
